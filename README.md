## 🎯 What I'm Learning (The 12-Week Sprint)

### Phase 1: Foundations & Networking (Weeks 1-3)

- **Linux Internals:** Abandoning the GUI for headless server management, process signals (`SIGTERM` vs `SIGKILL`), and the octal permission system.
    
- **Text Processing:** Mastering the "LeetCode for Ops" (`grep`, `awk`, `sed`) to extract truth from gigabytes of logs.
    
- **Networking Layers 3, 4, & 7:** CIDR notation, TCP/UDP port troubleshooting, and DNS resolution.
    
- **Automation:** Writing "the glue" with Bash scripting.
    

### Phase 2: The Manual Cloud & Containers (Weeks 4-6)

- **AWS "The Hard Way":** Manually architecting custom VPCs with Public/Private subnets, NAT Gateways, and Security Group chaining.
    
- **Identity & Access Management:** Implementing the "Principle of Least Privilege" and securing the Root account.
    
- **Docker:** Packaging code to solve "it works on my machine" using multi-container Docker Compose environments.
    

### Phase 3: Infrastructure as Code & CI/CD (Weeks 7-9)

- **GitHub Actions:** Building quality-gated CI pipelines with automated linting and security scanning.
    
- **Terraform:** Shifting from manual clicking to declarative Infrastructure as Code (IaC).
    
- **Kubernetes:** Orchestrating self-healing applications using Pods, Deployments, and Services in Minikube.
